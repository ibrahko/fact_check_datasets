# Build APK avec EAS Build

## Prérequis

- Node.js installé (version LTS recommandée)
- Expo CLI / EAS CLI disponible
- Un compte Expo (connecté sur le CLI)

## Commandes

```bash
npm install -g eas-cli
eas login
eas build --platform android --profile preview
```

## Téléchargement de l'APK

Une fois le build terminé, l'APK est disponible depuis le dashboard Expo :

- https://expo.dev/accounts/ibrahko
